# Library Management System

## Features
- User registration and login.
- Admin dashboard to manage books and users.
- User dashboard to search and borrow books.
- Transaction history tracking.

## Setup Instructions
1. Clone or download the project.
2. Import `library_management.sql` from the `db/` folder into your MySQL database.
3. Update the database credentials in `php/config.php`.
4. Run the project using a local server (e.g., XAMPP).

## Folder Structure
- `css/`: Contains stylesheets.
- `js/`: Contains JavaScript files for interactivity.
- `php/`: Contains server-side scripts for database interactions.
- `db/`: Contains SQL scripts for database setup.
